## Virtual DOM
Manipulating DOM might have side effects: * Reading some DOM properties can trigger reflows. * Setting some DOM properties also triggers recalculations. * Setting multiple values per update cycle can lead to several re-calculations.

Async / Await

function search(query) {
  return fetch(`/search/?q=$(query)`)
         .then(response => response.json())
         .then(json => json.results.map(...))
         .catch(reason => console.log(reason));
}
async function search(query) {
  try {
    let response = await fetch(`/search/?q=${query}`);
    let json = await response.json();
    return json.results.map(...);
  }
  catch (reason) {
    console.log(reason);
  }
}
